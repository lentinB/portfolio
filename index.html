Обо мне

 

Здравствуйте, меня зовут Валентин. Я начинающий WEB разработчик. Есть возможность работать удаленно.
Знания html и css. В качестве среды разработки использую PhpStorm. Знаком и использую на практике Git.
Владею навыком верстки сайтов с Psd макетов. В дальнейшем интегрирую вёрстку c CMS Wordpress.
Вёрстку делаю адаптивную, кроссбраузерную. Для написания Css использую препроцессор Sass(Scss). Собираю проект с помощью сборщика Gulp. В качестве менеджера пакетов использую Bower.
Постоянно изучаю новые технологии, открываю для себя интересные библиотеки и фреймворки. В данное время изучаю JS и PHP.
